# Vim notes

## Setup

**Install plugin manager**
```
mkdir -p ~/.vim/autoload
cd ~/.vim/autoload
curl -O https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

**Add to .vimrc to load**
```
:so (path to repo)/bashstuff/vim/vimrc.vim
```

**vim-plug commands**
In vim:-
```
:PlugStatus
:PlugInstall
```
